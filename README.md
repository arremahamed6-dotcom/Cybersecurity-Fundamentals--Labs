# Cybersecurity Fundamentals Labs

This repository contains hands-on labs and projects completed as part of the
Cybersecurity Fundamentals course.

## Course Objectives
- Understand the CIA Triad (Confidentiality, Integrity, Availability)
- Identify common malware types and attack vectors
- Learn basic authentication and access control concepts
- Understand foundational networking and security principles
- Analyze system logs for security-related events

## Labs Included
- File encryption and hashing (CIA Triad)
- Malware analysis using safe test files
- Password strength and least privilege testing
- Network traffic analysis with Wireshark
- Windows Event Viewer log analysis

## Tools Used
- Windows Event Viewer
- PowerShell
- Wireshark
- VirusTotal
- 7-Zip
- Sysinternals Suite
- Ubuntu Linux (Virtual Machine)

## Disclaimer
All labs in this repository are for educational purposes only.
